# AP242 ed3 (Managed Model-based 3D Engineering - Edition 3)

AP242 ed3 (ISO 10303-242:2022) は、従来の ed2 をベースにしつつ、デジタルスレッドの更なる強化を目的とした最新のリリースです。

## 1. 主な進化点
- **電気ハーネス (Electrical Harness)**: 電子機器設計データとの統合がより強固に。
- **付加製造 (Additive Manufacturing)**: 3Dプリント特有のプロセス情報や積層パラメータの管理。
- **MBSE (Model Based Systems Engineering)**: SysMLなどのシステムモデルとの連携。
- **長期保存 (LOTAR)**: ライフサイクル全体を通じたアーカイブ機能の拡張。

## 2. 実装上の注記
ed2 との高い互換性が維持されていますが、以下の新しいエンティティ群が追加されています。
- **Additive Manufacturing**: `PRINTED_PART_DEFINITION` など。
- **Harness**: ケーブル、コネクタ、ルーティング情報の拡張。

## 3. 互換性について
ed3 対応を謳っているCADベンダーは増えていますが、高度な電気系データやAMデータは、CAx-IFの推奨プラクティスに従った相互運用性テストが進行中の段階です。
