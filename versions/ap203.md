# AP203 (Configuration Controlled Design)

AP203は、STEPの中でも最も古くから広く普及している標準の一つです。

## 1. このAPは何のためにあるか
* 3D設計の形状管理
* 基本的なアセンブリ構造の定義

## 2. できること（保証される）
* 3D幾何形状（B-rep）
* 基本的なプロダクト管理情報（ID、名称など）

## 3. できない／保証されないこと
* **色・レイヤの保持** (e1の場合)
* PMI
* テセレーション形状

## 4. 向いている用途
* 非常に古いシステムとのデータ交換
* 形状のみが重要な場合
