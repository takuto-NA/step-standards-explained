# AP242 ed1 vs ed2

AP242にはEdition（版）があり、Edition 2で大幅な機能拡張が行われました。

## 主な違い

| 機能 | Edition 1 (2014) | Edition 2 (2020) |
| :--- | :--- | :--- |
| PMI | 基本的なサポート | セマンティックPMIの強化 |
| 電気配線 | ❌ | ✅ |
| 3D積層造形 | ❌ | ✅ |
| テセレーション | 基本機能 | 強化 |

## 実装上の注意
現在、多くのCADがAP242をサポートしていますが、その中身がed1かed2かは実装によります。エクスポートオプションを確認してください。
