# AP214 (Core Data for Automotive Design)

AP214は、自動車業界の要求に応える形で開発され、現在でも広く使われている標準です。

## 1. このAPは何のためにあるか
* 自動車設計における複雑なデータの交換
* 意匠（色・レイヤ）やアセンブリの高度なサポート

## 2. できること（保証される）
* 3D B-rep 形状
* **色・レイヤー情報**
* 複雑なアセンブリ構造
* テキスト注釈（非セマンティック）

## 3. できない／保証されないこと
* 高度なセマンティックPMI（幾何公差の意味内容など）
* 最新のMBDワークフローへの完全対応

## 4. 向いている用途
* 一般的な機械設計データの交換
* 「色が消えると困る」場合

## 5. 主要エンティティ
自動車設計で重視される表示プロパティに関連するエンティティです。

- **PRESENTATION_LAYER_ASSIGNMENT**: レイヤ情報の管理。
- **PRESENTATION_STYLE_ASSIGNMENT**: 形状とスタイルの紐付け。
- **COLOUR_RGB**: RGB値による色定義。
- **NEXT_ASSEMBLY_USAGE_OCCURRENCE**: アセンブリ構造。
- **STYLED_ITEM**: ジオメトリへのスタイル一括適用。
