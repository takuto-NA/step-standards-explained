# AP242 (Managed Model-based 3D Engineering)

AP242は、AP203とAP214を統合し、現代的な製造プロセスに対応させた最新のSTEP標準です。

## 1. このAPは何のためにあるか
* 航空宇宙・自動車業界の要求を統合
* MBD (Model Based Definition) の推進
* 長期データ保存 (LOTAR) への対応

## 2. できること（保証される）
* 3D幾何形状（B-rep, CSG）
* アセンブリ構造
* **PMI (Product Manufacturing Information)**
  * グラフィカル（表示用線分）
  * セマンティック（意味のあるデータ）
* テセレーション（軽量形状）
* 複合材構造など

## 3. できない／保証されないこと
* CAD固有のヒストリ（パラメータ）
* 非常に特殊なベンダー固有プロパティ

## 4. よくある誤解
* 「AP242ならPMIが完璧に伝わる」
  * 現実にはセマンティックPMIの解釈はCADベンダーの実装（CAx-IF推奨）に依存します。

## 6. 主要エンティティ
実装時に意識すべき主要なエンティティ群です。

- **GEOMETRIC_TOLERANCE**: 位置公差、平面度などの基底クラス。
- **SHAPE_ASPECT**: 形状の特定部位（面、穴）に「データム」等の意味を付与。
- **NEXT_ASSEMBLY_USAGE_OCCURRENCE (NAUO)**: アセンブリの階層構造を定義。
- **DIMENSIONAL_SIZE / LOCATION**: 公差を伴う寸法情報。
- **CONTEXT_DEPENDENT_SHAPE_REPRESENTATION**: 配置行列の管理。
